Objet du site

Application web statique (HTML, CSS, JavaScript) pour la veille et la prévention du risque terrorisme, à l’usage des équipes en accueil du public et des agents de sécurité. Elle complète les consignes employeur et les référentiels officiels ; elle ne remplace pas une formation diplômante ni les textes de loi à jour.

Important : n’ouvrez pas index.html en double-clic (file://) pour une démo complète — utilisez un petit serveur HTTP (voir ci-dessous).

Démarrage (serveur HTTP)

Avec Node.js (recommandé)

Dans le dossier du projet (celui qui contient package.json) :

npm install
npm start

Le terminal affiche une URL du type http://127.0.0.1:8080/ (ou un port suivant si 8080 est pris). Ouvrez-la dans le navigateur.

Sans Node.js

python3 -m http.server 8080 --bind 127.0.0.1

Puis ouvrez http://127.0.0.1:8080/ manuellement.

Carte

  • Recherche : champ en barre au-dessus de la carte.
  • Couches : panneau gauche (bouton ◀ sur grands écrans ; menu ☰ sur mobile).
  • Calque tensions : activable depuis le panneau « Couches », pas depuis la barre supérieure.
  • Plein écran : bouton sur la pile d’outils à droite de la carte (icône ⛶ ; × pour quitter), ou touche F hors champ de saisie.
  • Zoom : boutons + / − sur la carte, ou raccourcis clavier + / − quand le focus n’est pas dans un champ.
  • Clic pays : fiche (modale) avec blocs contexte / sécurité selon les données disponibles.

Savoir & Terrorisme

Savoir regroupe plusieurs rubriques en sous-onglets. La zone France · terrorisme propose notamment une chronologie compacte (fichier JSON partagé avec l’onglet Terrorisme).

Terrorisme détaille le cadre (France, UE, ONU), les liens utiles et le tableau chronologique France sur cinq colonnes.

Quiz, références & guide

  • Quiz : banque de questions indicatives ; croiser avec les sources officielles.
  • Références : liens vers sites primaires (Pew, ONU, Intérieur, etc.).
  • Guide : diaporama et textes pédagogiques intégrés à la même application.

Build dist/ & vérification

Pour produire un dossier minimal prêt à héberger (Pages, S3, clé USB avec serveur local) :

npm run build:html

Cette commande exécute d’abord node scripts/verify.mjs, puis copie index.html, pedagogie.html, sw.js, assets/ et docs/ (dont ce manuel) dans dist/.

Contrôle rapide des scripts JS : npm run verify ou node scripts/verify.mjs.
