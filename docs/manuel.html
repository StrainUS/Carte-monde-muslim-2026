1. Présentation

L'application Islam mondial 2026 agrège sur une interface unique une carte des sensibilités religieuses, des fiches pays horodatées, un guide pédagogique et un quiz d'auto-évaluation. Elle est conçue pour être consultée depuis un poste fixe, une tablette, ou un appareil mobile, avec ou sans connexion réseau.

Avertissement. Cet outil ne remplace ni la formation réglementaire (CNAPS, SSIAP…), ni les consignes de l'employeur, ni les publications officielles à jour. Le quiz est un outil d'auto-évaluation sans valeur certifiante.

2. Modules de l'application

Carte

Couches sunnite / chiite / tensions régionales / hotspots terroristes. Clic pays → fiche détaillée.

Savoir

Timeline historique, fiches courants, glossaire, portails officiels.

Terrorisme

Chronologie France, hotspots, repères de vigilance.

Guide

Diaporama pédagogique (diversité, sunnisme, chiisme, ibadisme).

Quiz

20 questions QCU/QCM d'auto-évaluation avec correction immédiate.

Sources

Bibliographie critique, accès direct aux textes primaires.

3. Prise en main de la carte

Navigation

  • Zoom : molette, pincement (trackpad ou écran tactile), ou boutons + / − à l'écran.
  • Déplacement : glisser-déposer avec la souris ou le doigt.
  • Recherche pays : champ en haut à gauche (autocomplétion ISO et nom).
  • Couches : barre latérale — cochez/décochez sunnites, chiites, tensions, hotspots.

Interprétation visuelle

  • Radars concentriques = tensions régionales. 1 onde = niveau 1, 3 ondes = niveau 3. Les niveaux inférieurs n'apparaissent qu'aux zooms élevés (dévoilement progressif).
  • Triangles ⚠ pulsants = hotspots terroristes. Cliquer pour afficher les repères de vigilance.
  • Teintes de remplissage = dominante religieuse (vert sunni, violet chiite, dégradés pour les minorités).

5. Fiches pays et sources

Chaque pays dispose d'une fiche détaillée accessible en cliquant sur son territoire ou via la recherche. On y trouve :

  • démographie religieuse et répartition par courant ;
  • notes de sécurité et repères de vigilance (si renseignés) ;
  • sources citées — liens directs vers les organismes officiels (Pew, INSEE, BAMF, Europol, USCIRF…) ;
  • date de fraîcheur (asOf) — mois/année de la dernière révision éditoriale.
Pour toute utilisation professionnelle, croiser au moins deux sources de nature différente (organisme international + source nationale) et vérifier la date de publication la plus récente disponible.

6. Utilisation hors ligne

L'application est une PWA (Progressive Web App) : après la première visite, elle reste disponible sans connexion.

  • Un badge apparaît en haut à droite dès que le réseau est indisponible.
  • La navigation entre pages, la carte et les fiches pays fonctionnent à partir du cache local.
  • En cas de blocage prolongé au chargement (> 6 s), un bouton Réinitialiser le cache et recharger purge le service worker.

Pour une copie autonome sur clé USB : cloner le dépôt, exécuter npm run build dans app/, puis copier le dossier app/build/. Il suffit d'ouvrir index.html via un serveur HTTP local (pas en file://).

7. Installation locale

Prérequis : Node.js ≥ 20, npm ≥ 10, git. 

git clone https://github.com/StrainUS/Carte-monde-muslim-2026.git
cd Carte-monde-muslim-2026/app
npm ci
npm run data:geojson
npm run dev

L'application est servie sur http://127.0.0.1:5173. Pour un build de production : 

npm run build
npm run preview    # http://127.0.0.1:4173

8. Déploiement

Le déploiement est automatique à chaque push sur main via GitHub Actions, vers GitHub Pages.

Le build est entièrement statique : il peut également être publié sur tout hébergement statique (Netlify, Vercel, S3, Nginx…) en copiant le contenu de app/build/.

9. Dépannage

La carte reste bloquée sur « Chargement de la carte… »

Cliquer sur Réinitialiser le cache et recharger (apparaît après 6 s de chargement anormal). Sinon, dans les outils développeur du navigateur, onglet Application → Service Workers → Unregister, puis recharger.

Les couches ne se mettent pas à jour immédiatement

Vider le cache du service worker (voir ci-dessus). La version ≥ 2.0 corrige ce comportement.

Erreur ENOENT package.json

Vous n'êtes pas dans app/. Lancer pwd et cd app.

10. Références officielles

La méthodologie complète et la liste exhaustive des organismes de référence sont documentées dans docs/SOURCES.md. Les repères opérationnels les plus courants en France :

  • Ministère de l'Intérieur — interieur.gouv.fr
  • Signalement radicalisation — stop-djihadisme.gouv.fr
  • Code pénal (section terrorisme) — legifrance.gouv.fr
  • Europol TE-SAT — europol.europa.eu
